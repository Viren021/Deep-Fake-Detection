## ⚠️ Important: Model File Required

Due to GitHub's file size limits (100MB), the pre-trained model file could not be uploaded to this repository. To run this project, you must download the model manually.

### Setup Instructions:
1. Download the `df_model.pt` file from this link:https://drive.google.com/file/d/19jdoB2CaogjAN0yXBz_zLCVrzhYn3sGF/view?usp=drive_link
2. Place the downloaded file inside the folder: `DeepFake_Detection/model/`
3. The project should now run correctly.
